## What the operator has to do

`npm publish` requires a **granular npm token** that only the operator
(Carter) holds. It is not baked into CI for security reasons. To cut
the release:

1. Tag the release on `main`:
   ```bash
   git fetch origin main && git checkout main && git pull --ff-only
   git tag -a v0.8.1 -m "sentinelayer-cli v0.8.1"
   git push origin v0.8.1
   ```
2. Trigger the `Release` workflow manually with `publish=true`:
   ```bash
   gh workflow run release.yml \
     --repo mrrCarter/create-sentinelayer \
     --ref v0.8.1 \
     -f publish=true
   ```
3. Watch the run to completion:
   ```bash
   gh run watch <run-id> --repo mrrCarter/create-sentinelayer --exit-status
   ```
4. Verify publish:
   ```bash
   npm view sentinelayer-cli version
   # → 0.8.1
   ```
